## Responsive Behavior

### Desktop (1400px+)
- Form container: 1400px max width
- Field layout: One column (or two if `formLayout: "two-column"`)
- Section headings: Full width
- Textareas: Full width
- Checkboxes: Full width

### Tablet (768px - 1399px)
- Form container: Full width with padding
- Field layout: Adjusts based on `formLayout`
- Sections span full width
- Good spacing maintained

### Mobile (< 768px)
- Form container: Full width
- All fields: Single column (auto-stacks)
- Sections: Full width
- Touch-friendly tap targets
- Readable font sizes
